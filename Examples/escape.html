There are 3 characters that need to be escaped in HTML

  • The less than character (<)
  • The greater than character (>)
  • And the ampercent sign (&)

These can be escaped by writing an ampercent sign followed by "lt" for less than, "gt" for greater than and "amp" for the ampercent sign, followed by a semi-colon.

ie, &lt; / &gt; / &amp;


there are other symbols that can be represented in HTML by referenceing syntax

for example, the copyright symbol (©) is represented by an ampercent followed by "copy" followed by semi-colon.
ie, &copy;

to represent a non breaking space, so that words between said space will always stay together use &nbsp;
